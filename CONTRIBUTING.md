# Contributing to TR Tier Tagger

Thank you for your interest! However, this repository has strict rules for contributions and forks.

## Rules for Forks and Modifications:

1. **No unauthorized modifications or redistributions** without explicit written permission from Lora Technologies.
2. Any changes you make **must** be submitted as a Pull Request (PR) to this official repository for review and approval.
3. If you fork this repository:
   - You **must** keep all original attributions (to Lora Technologies and original TierTagger developers).
   - You **must** provide full source code of your modified version publicly.
   - You **must not** reupload modified versions to platforms like CurseForge/Modrinth without prior approval.
   - Commercial use or claiming authorship is strictly prohibited.

## Violations

Violations may result in:
- DMCA takedown requests to hosting platforms
- Reports to original licensors (MPL-2.0 violations)
- Legal action if applicable
